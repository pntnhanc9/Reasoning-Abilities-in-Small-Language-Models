# 📊 BENCHMARK INFORMATION: Self-Steering Language Models (DisCIPL)

## 📁 Project Overview

This repository contains two implementations of Self-Steering Language Models:

1. **Original DisCIPL** (`self-steering/`): Full implementation from arXiv:2504.07081
2. **SimplifiedDiSCIPL** (`true_working/`): Cost-efficient, GPU-friendly version

---

## 🏗️ IMPLEMENTATION DETAILS

### Original DisCIPL (self-steering/)
- **Architecture**: Self-Steering Language Models using DisCIPL framework
- **Planner**: OpenAI GPT-4o API ($10/query) + LLaMPPL program generation
- **Follower**: Sequential Monte Carlo (SMC) with 32 particles
- **Models**: GPT-4o, Llama models (3.1-8B, 3.2-1B)
- **Framework**: llamppl, transformers, torch
- **GPU Memory**: 64GB peak
- **Cost**: $600-10K per 1000 tasks
- **Speed**: 30-60 seconds per task

### SimplifiedDiSCIPL (true_working/)
- **Architecture**: Simplified planner-follower pipeline
- **Planner**: Local Qwen 3.5 9B AWQ 4bit model (free)
- **Follower**: 2 parallel followers using Qwen 3.5 0.8B
- **Models**: cyankiwi/Qwen3.5-9B-AWQ-4bit, Qwen/Qwen3.5-0.8B
- **Framework**: transformers, torch (local inference only)
- **GPU Memory**: 18GB peak
- **Cost**: $0 (open-source models)
- **Speed**: 20-40 seconds per task

## 📁 RESULT STRUCTURE

Results saved to `results/<timestamp>/`:
```
results/
├── simplified_disciple_20260504_120000/
│   ├── summary.json          # Overall statistics
│   ├── task_0000/result.json # Individual task results
│   └── task_0001/result.json
└── disciple_experiment_20260504_120000/
    ├── results.json          # All results
    ├── summary.json          # Statistics
    └── log.txt              # Execution log
```

Each result.json contains:
```json
{
  "task_id": 0,
  "task_prompt": "...",
  "strategy": ["Step 1", "Step 2", ...],
  "generated_answer": "...",
  "is_valid": true,
  "num_attempts": 1,
  "total_time": 25.3,
  "evaluation": {"valid": true}
}
```

## 🔧 CONFIGURATIONS

### Model Configurations (true_working/config_examples.py)
- `config_minimal()`: Fastest, least memory
- `config_balanced()`: Default, recommended
- `config_high_quality()`: Best quality
- `config_low_memory()`: Limited VRAM
- `config_float32()`: Better precision
- `config_research()`: No constraints

### Command Line Options
- `--num-tasks N`: Number of tasks
- `--max-tokens N`: Max generation tokens
- `--max-attempts N`: Max retry attempts
- `--dtype float16|float32`: Data type
- `--results-dir PATH`: Output directory
